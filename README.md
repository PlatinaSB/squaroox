# squaroox
squaroox is a small project for semester project 

# to run the website 

## requirements
1. node 18.17+
2. golang 1.21+

## prep

1. clone the repo 
```bash
git clone https://github.com/PlatinaSB/squaroox.git
```

```bash
cd squaroox
```


2. install dependecy 
```bash 
npm install 
```

3. build the website
```bash
npm run build
```

## run the server / website

```bash
go run ./main.go
```


# developing the website
```bash
npm run dev -- --open --host
```
